CSS Grid is a layout system in CSS for creating two-dimensional layouts (both rows and columns).

  • Grid Container: The element where display: grid is applied.
  • Grid Items: Direct children of the grid container.
  • Grid Lines: The lines that divide the grid.
  • Grid Tracks: Rows and columns between grid lines.
  • Grid Cell: A single unit in the grid.
  • Grid Area: A rectangular area spanning one or more cells.
